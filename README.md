# AI-Assisted Tourism Recommendation and Booking System for Gilgit-Baltistan

This repository contains the Final Year Project (FYP) developed for the BS Computer Science program.

## Project Overview
The project aims to provide a centralized, cloud-based tourism platform for Gilgit-Baltistan. It integrates hotels, tour guides, transport providers, and tourism activities into a single system with AI-assisted recommendations.

## Technology Stack
- Frontend (Web): ReactJS
- Mobile App: Flutter
- Backend: Python (FastAPI)
- Database: Firebase
- Cloud Deployment: Firebase / Cloud Hosting

## Repository Structure
- `backend/` – API and business logic
- `web-app/` – Web application
- `mobile-app/` – Mobile application
- `docs/` – Proposal and documentation
